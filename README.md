# Field Journal

A one-file journal for a summer job: **quotes**, **questions**, and **notes**, styled as ruled index cards in a card box.

- Entries save to the browser's localStorage — nothing ever leaves the device.
- Export/Import buttons in the footer move cards between devices as a JSON file.
- "Pull a card" shows a random entry (answers hidden until revealed) for self-quizzing.
- Works offline after the first visit (service worker).

No build step. `index.html` is the whole app; host it anywhere static.
